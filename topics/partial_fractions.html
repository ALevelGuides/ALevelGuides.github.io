Partial Fraction Decomposition

Consider the following process:

\[\begin{aligned} &\frac{2}{2x-1} + \frac{1}{x+1} = \frac{2(x+1) + (2x-1)}{(2x-1)(x+1)} = \frac{4x+1}{(2x-1)(x+1)} \end{aligned}\]

It is easy to go from the left hand side to the right hand side by simple arithmetic. However going from the right to the left requires partial fraction decomposition. Here we will learn how to do this.

We can do this by guessing the decomposition with arbitrary constants \(A\) and \(B\).

\[\frac{4x+1}{(2x-1)(x+1)} = \frac{A}{2x-1} + \frac{B}{x+1}\]

Now we can multiply both sides by \((2x-1)(x+1)\) to get:

\[4x+1 = A(x+1) + B(2x-1)\]

Notice this is an identity, meaning it must hold for all values of \(x\), so we can strategically choose values of \(x\) that make the terms \(0\), to find the values of \(A\) and \(B\).

\[\begin{aligned} &x = -1 \implies -3 = -3B \implies B = 1 \\ \\ &x = \frac{1}{2} \implies 3 = \frac{3A}{2} \implies A = 2 \end{aligned}\]

which gives us the decomposition we wanted.

However, things get more complicated if we have a quadratic or a higher degree term in the denominator. For example consider:

\[\frac{4x+2}{(x^2 + 1)(x - 1)}\]

While doing the decomposition, the numerator will be a polynomial of degree one less than the denominator, so the numerator of the term with the quadratic denominator will be a linear term.

\[\frac{4x+2}{(x^2 + 1)(x - 1)} = \frac{Ax+B}{x^2 + 1} + \frac{C}{x-1}\]

We can now do the same thing we did before, but the equation we get will be more complicated.

\[4x+2 = (Ax+B)(x-1) + C(x^2 + 1) = Ax(x-1) + B(x-1) + C(x^2 + 1)\]

We can find the values of the coefficients by putting in test values.

\[\begin{aligned} &x = 1 \implies 6 = 2C \implies C = 3 \\ \\ &x = 0 \implies 2 = -B + C = -B + 3 \implies B = 1 \\ \\ &x = 2 \implies 10 = 2A + B + 5C = 2A + 16 \implies A = -3 \end{aligned}\]

which is the decomposition we wanted

\[\frac{4x+2}{(x^2 + 1)(x - 1)} = \frac{-3x+1}{x^2 + 1} + \frac{3}{x-1}\]

We can do something interesting when there is a power of a polynomial as one of the factors in the denominator. For example consider the following rational function:

\[\frac{2x^2 + x - 1}{(x + 1)^2(x-2) }\]

The \((x + 1)^2\) term is technically a quadratic, so we would need the term \(Ax+B\) in the numerator of its partial fraction.

\[\frac{2x^2 + x - 1}{(x + 1)^2(x-2) }\ = \frac{Ax + B}{(x+1)^2} + \frac{C}{x-2}\]

However, it's not too difficult to show that we can also decompose it in the way:

\[\frac{2x^2 + x - 1}{(x + 1)^2(x-2) }\ = \frac{A}{x+1} +\frac{B}{(x+1)^2} + \frac{C}{x-2}\]

Doing it this way is usually easier to solve. So essentially, when you have a power of a polynomial in a denominator, you'll have to write all the powers of the term up to it while decomposing. Here are a couple examples of decompositions in all cases:

\[\begin{aligned} &\frac{2x^3 + 3x^2 - 1}{(x+1)^3(x-2)} = \frac{A}{x+1} + \frac{B}{(x+1)^2} + \frac{C}{(x+1)^3} + \frac{D}{x-2} \\ \\ &\frac{2x - 1}{(x^2 + 2x - 1)^2(2x - 3)} = \frac{Ax+B}{x^2 +2x - 1} + \frac{Cx+D}{(x^2 + 2x - 1)^2} + \frac{E}{2x-3} \end{aligned}\]

Using high powers like this will make the question significantly more annoying, which is why you won't usually be asked to do a partial fraction decomposition with the degree of the denominator higher than 3, but it's still useful to know about.

Finally, another situation that may happen is when the degree of the numerator is higher than the degree of the denominator, in this case we have to perform polynomial division and then do the decomposition

\[\begin{aligned} &\frac{x^3 + x - 1}{(x+1)(x-1)} = \frac{x(x+1)(x-1) + 2x-1}{(x+1)(x-1)} \\ \\ &=x + \frac{2x-1}{(x+1)(x-1)} = x + \frac{A}{x+1} + \frac{B}{x-1} \end{aligned}\]

Problem 1:Decompose the following into partial fractions

\[\frac{12}{(2-x)(1-2x)^2}\]
Solution

We can decompose in the form

\[\frac{12}{(2-x)(1-2x)^2} = \frac{A}{2-x} + \frac{B}{1-2x} + \frac{C}{(1-2x)^2}\]

Expanding, we get

\[\begin{aligned} &12 = A(1-2x)^2 + B(2-x)(1-2x) + C(2-x) \\ \\ &x = 2 \implies 12 = 9A \implies A = \frac{4}{3} \\ \\ &x = \frac{1}{2} \implies 12 = \frac{3C}{2} \implies C = 8 \\ \\ &x = 0 \implies 12 = A + 2B + 2C = \frac{4}{3} + 16 + 2B \implies B = -\frac{8}{3} \\ \\ &\frac{12}{(2-x)(1-2x)^2} = \frac{\frac{4}{3}}{2-x} + \frac{\frac{-8}{3}}{1-2x} + \frac{8}{(1-2x)^2} \end{aligned}\]

which is the expansion we wanted

This basically covers all possible cases of partial fraction decomposition, but you still may ask why or when we would need to use it. It turns out that it is incredibly useful in several areas, such as integrals, telescoping series, binomial expansions and more.

Problem 2:Find the antiderivative:

\[\int \frac{4x+2}{(x^2+1)(x-1)}dx\]
Solution

We can use the partial fraction decomposition we did before to write the integral as

\[\int \frac{4x+2}{(x^2+1)(x-1)}dx= \int (\frac{-3x+1}{x^2 + 1} + \frac{3}{x-1})dx\]

Now we can do some manipulation to simplify this integral

\[\begin{aligned} &\int (\frac{-3x+1}{x^2 + 1} + \frac{3}{x-1})dx = -3\int \frac{x}{1+x^2}dx + \int \frac{1}{1 + x^2}dx + 3\int \frac{1}{x-1}dx \\ \\ &= -\frac{3}{2} \int \frac{2x}{1+x^2}dx + \int \frac{1}{1 + x^2}dx + 3\int \frac{1}{x-1}dx \\ \\ &= -\frac{3}{2}\ln{|1+x^2|} + \tan^{-1}{x} + 3\ln{|x-1|} + C \end{aligned}\]

which is our solution. Note that this kind of integral will probably not be asked in an A-Level Maths exam, as the integral of \(\frac{1}{1+x^2}\) is not in the specification.
